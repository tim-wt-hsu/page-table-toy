Page Table Toy: RISC-V Virtual Address Translation Visualizer

This interactive tool allows users to explore how RISC-V systems translate virtual addresses to physical addresses using page tables.

Supported RISC-V Paging Modes:

  • Sv32: 32-bit virtual addressing with 2-level page tables.
  • Sv39: 39-bit virtual addressing with 3-level page tables.

Key Features:

Step-by-step page table walk, Interactive PTE (Page Table Entry) modification, Physical memory viewport visualization, support for Huge Pages (Gigapages and Megapages).
